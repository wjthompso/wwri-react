About the Index

Why an Index?


An index allows information from diverse datasets to be brought into a shared decision space, so everyone can evaluate resilience using a consistent framework.


Wildfire resilience is inherently multi-dimensional, spanning ecological, social, and infrastructural factors that are often managed in separate silos.


The WRI combines multiple indicators into a composite score so users can interpret resilience systematically, compare outcomes across regions, and benchmark progress over time.


Because the index is spatially explicit and temporally trackable, it supports adaptive management, long-term planning, and coordinated action to help communities and landscapes live sustainably with fire.
